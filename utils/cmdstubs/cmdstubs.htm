CmdStubs

This utility script helps you invoke Softimage commands within your C++ API-base plug-in. It generates functions for each Softimage Command (including any custom commands installed on your system). These functions are easier to use than calling Application::ExecuteCommand directly.

  1. Run the JScript in Softimage. This generates the cmdstubs.cpp and cmdstubs.h file.

  2. Include the resulting cmdstubs.cpp and cmdstubs.h in your C++ plug-in. You could also extract just the code related to the commands you want to call.

  3. Softimage commands can now be invoked by calling a function. See the dotXSIConverter example for an example usage.

  4. When each new version of Softimage ships you can run this script again to include any new commands in your project.

Please refer to the script file (cmdstubs.js) for further details.
