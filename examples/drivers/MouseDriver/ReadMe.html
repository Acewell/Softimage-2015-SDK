Mouse Driver

This example provides the source code for the mouse driver included with Softimage. The mouse driver uses the Softimage object model to drive the X and Y position of a selected object in Softimage.

Files

mousedriver.vcproj Msdev project
mousedriver.cpp Implementation of the mouse device driver in C++ using the Softimage Object Model.
mousedriver.def Module-definition (.def) file that lists the exported functions.
stdafx.cpp stdafx.h Precompiled header file
update.bat Copies the updated DLL to %XSI_HOME%\Application\bin. Note that this overwrites the shipped version of the mouse driver.
mouseexample.xsidevice Device description file that specifies the device driver and its channels. Softimage loads device description files when a user activates the Device Manager. For more information, look up ".xsidevice" in the index of the Softimage SDK documentation.

Running the Example

To run the example:

  1. In the Animation toolbar, click Tools > Devices > Device Manager.
  2. In the Device Manager property page, select the Device Manager Active check box.

  3. Click Add and select Mouse from the list of device drivers.

    The Device Manager: Mouse property page appears. The mouse driver uses a default preset (specified in the .xsidevice file) that drives the X and Y position of the current selection.

  4. Activate the mouse driver.
  5. Select an object (such as a null or a sphere) and move the mouse pointer to change the position of the selected objects.

Building the Mouse Driver

To build the mouse driver on Windows

  1. Open an Softimage command prompt, and type devenv to start Visual Studio .NET.

    Starting Visual Studio .NET from an Softimage command prompt ensures that environment variables such as XSISDK_ROOT are set (otherwise you'll get build and link errors).

    Tip To load the mousedriver project from the command line, type:

    devenv mousedriver.vcproj

  2. If you did not load the project from the command line:
    • In Visual Studio .NET, open mousedriver.vcproj.
  3. Select a configuration (Debug Win32, Release Win32, Debug x64 or Release x64) and build the DLL.

Installing the Mouse Driver

To install the mouse driver

  • Copy mouseexample.dll/so to Application\bin in the Softimage installation folder.
  • Copy mouseexample.xsidevice to Data\Devices in the Softimage installation folder.
