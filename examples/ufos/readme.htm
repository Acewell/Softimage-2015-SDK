UFO examples

This folder contains Media Illusion User Function Object plug-ins that are used inside the Softimage FX Tree.

How to Compile and install the ufo examples plug-ins

Windows

Compiling

Open any examples project with MSDEV, select a compilation target (debug or release) and compile. 

Installing

To install the result dll, simply run install_ufos.bat under the ufo folder from a Softimage command prompt.

Linux

Before running any command, you have to set up your Softimage environment. To achieve this, run the following command in a shell (tcsh) :

    source [Install dir]/.xsi_[version]

Compiling

In the shell (tcsh),go to the ufo folder and issue the following command:

    gmake

Cleaning

In the shell (tcsh), issue the following command:

    gmake clean

Installing

To install the result dso, simply from the tcsh shell, run ./install_ufos.csh under the ufo folder.
